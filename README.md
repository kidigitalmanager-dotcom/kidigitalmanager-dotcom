## Hey, I'm Leon

Founder & solo builder of **[UseEasy](https://github.com/kidigitalmanager-dotcom/useeasy)** — an AI-powered email classification platform serving 13 industry verticals.

I designed and built the entire system from scratch: an **11-gate classification pipeline** that combines deterministic rule matching with LLM validation, achieving **99.6% accuracy** across 2,730+ production emails. 131 active pack rules across e-commerce, real estate, logistics, B2B sales, coaching, hotel, telecom, education, manufacturing, marketing, finance, and energy. The stack runs on AWS Lambda (Node.js 20), PostgreSQL 16, and AWS Bedrock (Claude) in eu-central-1 for GDPR compliance.

### What I build

- **Hybrid AI pipelines** — deterministic-first, LLM-second. 80% of classifications never touch the LLM.
- **Multi-tenant SaaS architecture** — 13 industry verticals, domain-polymorphic labels, provider-agnostic spreadsheet engine, dual-provider email support (Gmail + Outlook).
- **Production integrations** — Gmail API, Microsoft Graph, Chrome Extension (MV3), Outlook Add-In, Stripe, n8n orchestration.
- **Security-first auth** — dual-algorithm JWT verification (ES256 + HS256), PII masking before LLM, timing-safe crypto.
